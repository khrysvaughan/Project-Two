Why Use This Site

Housing is one of the basic necessities of life. Everyone needs a home to live in. So buying a home is one of those life experiences that is rewarding, but can be stressful.
Cost is one of those reasons.
Where is the best place to buy a home? Using this site can help you answer that question.

Coding Approach

  • Create tables in mySQL and populate from CSV table import.
  • Use mysql connector then Flask to define routes for individual end points.
  • Use JavaScript to apply logic to the data.
  • Use Plotly to chart the data.
  • Use index.html or similar pages to host in the webpage.

    • Data Munging

  • 3 CSV files are merged into excel which were imported to MySql.
  • Data is then pulled out using python mysql connector and data is organized and converted from tuples to dictionaries
  • Data is jsonified out via Flask app route
  • JSON data is used for each plot/chart constructed
